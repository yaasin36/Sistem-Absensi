🌟 SPACE - Sistem Presensi Digital
SPACE (Sistem Presensi Digital) adalah aplikasi absensi karyawan modern berbasis Single Page Application (SPA) Serverless yang memadukan deteksi wajah cerdas, pelacakan lokasi (geolocation), dan ekosistem Google Workspace.

✨ Fitur Utama
Deteksi Wajah Real-time: Validasi presensi klien menggunakan face-api.js untuk memastikan wajah terekam jelas sebelum absen.

Live Geolocation & Watermark: Pencatatan otomatis koordinat lokasi (GPS), waktu, dan nama yang ditempelkan langsung pada foto bukti absensi.

Arsitektur SPA murni: Navigasi antar halaman sangat cepat dan mulus tanpa refresh browser.

Role-Based Access Control (RBAC): Pemisahan hak akses menu dan data untuk tingkat SuperAdmin, Admin, dan User.

Serverless Database: Tidak memerlukan server backend konvensional; sepenuhnya menggunakan Google Apps Script (REST API) dan Google Sheets.

Manajemen Izin: Pengajuan izin sakit dilengkapi dengan fitur unggah Surat Dokter yang otomatis masuk ke Google Drive.

🛠️ Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript, Bootstrap 5.3

AI Library: Face-API.js

Backend & Database: Google Apps Script, Google Sheets, Google Drive

Hosting: GitHub Pages
